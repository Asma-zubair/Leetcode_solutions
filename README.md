
# 📘 LeetCode Solutions Repository

This repository contains my solutions to selected LeetCode problems.  
Each problem has its **own folder**, and every folder contains exactly two files:

1. **README.md** — The problem’s full description (question only)  
2. **solution.py** — My Python implementation

This structure keeps each problem organized, simple, and easy to navigate.

---

## 📂 Repository Structure Example

```
leetcode-solutions/
│
├── 3583-count-special-triplets/
│ ├── README.md # problem description
│ └── solution.py # python solution
│
├── 0000-sample-problem/
│ ├── README.md
│ └── solution.py
│
└── README.md # (this file)
```

## ✔ Purpose

This repository is a clean collection of LeetCode problems and their Python solutions, organized in a simple and consistent folder structure.
